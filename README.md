# ClearwaterOS x86

ClearwaterOS x86 is an Arch Linux based live ISO. Special support for old Macs with broadcom wifi chip unsupported by Arch. The primary reason for making this was inspired by getting my macbook pro 13inch from 2015 to run archbased linux, but I hope to expand this to a workable linux distro for many older x86 computers and eventually arm based machines. The ultimate goal is a simple arch based os that can help bring old hardware back to life.
